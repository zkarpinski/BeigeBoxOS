# Feature Ideas — Windows 98 Desktop

## High Impact (core Win98 feel)

- [x] **System tray with live clock** — Real time updating every second, tooltip with full date. Volume icon opens a small slider popup.
- [x] **Right-click context menu on desktop** — "Arrange Icons", "Refresh", "New", "Properties" (opens Display). One of the most instinctive things people try.
- [x] **Start > Run dialog** — Type `calc`, `notepad`, `minesweeper` to open apps. Also opens URLs in Navigator.
- [ ] **Solitaire / FreeCell / Hearts** — Most-played Win98 apps. Solitaire especially screams authenticity.
- [x] **My Computer window** — Shows C:\, A:\ floppy, CD-ROM. Clicking a drive opens a fake folder view. Double-click files to open apps (e.g. My Resume.doc → Word 97).

## Medium Impact (adds depth)

- [ ] **Sound effects** — Window open/close, error, maximize. `chord.wav`, `ding.wav`, `tada.wav`.
- [ ] **Screensaver** — Flying Windows, 3D Pipes, Starfield, Mystify after idle. Controlled via Display Properties > Screensaver tab.
- [ ] **Dial-up connection sound** — When Navigator/IE5 loads a page, play modem handshake sound first with a "Connecting..." dialog.
- [x] **More Control Panel applets**
  - Date/Time — set the system clock (affects tray clock)
  - Sounds — assign sounds to system events
  - Mouse — pointer speed, double-click test (jackrabbit animation)
  - System — fake "Pentium II 400MHz · 128MB RAM · Windows 98 SE"
- [ ] **Taskbar right-click** — "Cascade Windows", "Tile Horizontally", "Tile Vertically", "Minimize All Windows", "Properties".

## Fun / Easter Eggs

- [x] **Windows Update** — Start > Windows Update → IE5 navigates to fake update page. "3 Critical Updates Found" → progress bar → "Please restart."
- [ ] **Random error dialogs** — Occasional fake Windows errors: "This program has performed an illegal operation", "MSVBVM60.DLL was not found."
- [ ] **ICQ or MSN Messenger** — Flower/UIN bubble icon and "Uh oh!" sound effect.
- [x] **Winamp visualizer** — Spectrum analyzer (75 bars, green gradient, peak markers) while "playing". Decays on pause/stop.
- [ ] **CD AutoPlay dialog** — "Windows has detected a new CD-ROM. What would you like to do?"
- [ ] **WinZip window** — Classic zipper mascot. Add a `.zip` file to the desktop.

## Quick Wins (polish)

- [ ] **Animated hourglass cursor** during boot/loading
- [ ] **Tooltips** on taskbar buttons (Win98 style black-on-yellow)
- [ ] **Desktop icon F2 rename** — click once to select, F2 to rename
- [ ] **More wallpapers** — Straw Mat, Carved Stone, Feathered Texture, classic tiled patterns
- [ ] **"About Windows 98" dialog** — From Start menu, shows Windows 98 logo and registration name
- [ ] **Low disk space notification** — Occasional "Drive C: is running low" balloon tip from tray
